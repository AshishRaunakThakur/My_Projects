# Ashmeera Empowers 🌟

**Empowering Abilities, Creating Opportunities**

Ashmeera Empowers is an inclusive employment platform that connects persons with disabilities with verified employers, mentors, and skill development opportunities.

## ✨ Features

### 👤 User Features
- Registration & Login with secure authentication
- Complete profile management (photo, resume, skills, education)
- Smart job search with filters (location, category, salary, experience)
- One-click job applications
- Save/Bookmark jobs for later
- Application history tracking
- Real-time notifications
- AI-powered job recommendations with match percentage

### 🏢 Employer Features
- Company profile creation and management
- Post, edit, and manage job listings
- View and manage applicants
- Shortlist, approve, or reject candidates
- Download resumes

### 🛡️ Admin Features
- Secure admin dashboard
- User and employer management
- Job and application oversight
- Analytics dashboard with Chart.js
- Report generation

### ♿ Accessibility Features
- Dark Mode / Light Mode
- High Contrast Mode
- Large Text Mode
- Text-to-Speech
- Speech-to-Text
- Keyboard Navigation
- WCAG 2.1 AA Compliant

### 🤖 AI Features
- Intelligent job recommendations based on:
  - User skills
  - Education background
  - Experience level
  - Disability type
- Match percentage display for each job

## 🚀 Tech Stack

- **Backend**: Python Flask
- **Database**: MySQL with SQLAlchemy ORM
- **Frontend**: HTML5, CSS3, Bootstrap 5
- **JavaScript**: Vanilla JS with Chart.js for analytics
- **Security**: Password hashing (bcrypt), CSRF protection, SQL injection prevention
- **Accessibility**: WCAG 2.1 compliant
